# Martin's Flying Machines - Modernized Archive

This repository contains the modernized and recovered archive of Martin Newell's "Flying Machines" website, originally hosted at `http://martin-newell.com/`.

## Project Overview

This site is a tribute to Martin Newell's incredible work in micro-radio-controlled aviation. The original content has been recovered from the Wayback Machine and redesigned with a modern, responsive, card-based layout.

### Key Features

*   **Modern Dashbord**: A new central hub for exploring the fleet.
*   **Card-Based Aircraft Pages**: Each plane has its own page with technical specifications, high-quality photos, and background stories.
*   **Media Recovery**: 
    - Full-resolution image galleries restored for all aircraft.
    - Legacy Flash videos replaced with modern HTML5 players.
    - PDF Plans and technical documents recovered and linked for download.
*   **Pure CSS/HTML**: No heavy frameworks, ensuring fast load times and longevity of the archive.

## Structure

*   `index.html`: The main dashboard.
*   `css/`: Stylesheets for the modern and legacy sections.
*   `planes/`: Individual aircraft project pages.
*   `pictures/`, `images/`: Original photographic assets.
*   `videos/`: Recovered flight videos (WMV format).
*   `plans/`: Technical drawings and templates.

## How to View

Simply open `index.html` in any modern web browser to explore the archive.
